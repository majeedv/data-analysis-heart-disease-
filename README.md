# Google Colab Data Analysis Project for Heart disease 

 Heart Disease Prediction & Analysis

This project is a **Data Analysis & Machine Learning project** on the UCI Heart Disease dataset.  
It covers data preprocessing, visualization, feature analysis, and predictive modeling using **Logistic Regression** and **Random Forest**.

## Features
- Exploratory Data Analysis (EDA) with **visualizations** for numeric and categorical features
- Data cleaning & missing value handling
- Target classification (binary: disease vs no disease)
- Model training and evaluation:
  - Logistic Regression
  - Random Forest
- 5-Fold Stratified Cross-Validation
- Feature importance analysis
- Confusion matrix and ROC curve visualization

##  Technologies
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn (LogisticRegression, RandomForestClassifier)
- Google Colab compatible

## Visualizations
- Distribution of numeric features
- Correlation heatmap
- Target vs feature plots
- Confusion matrix & ROC curves
- Feature importance bar chart
